# CSV Processing

Parsing CSV (Comma-Separated Values) files seems easy until you encounter a field that contains a comma inside quotes (e.g., `"Smith, John", 42`). A simple `strings.Split(line, ",")` will instantly break your application.

The `encoding/csv` package handles all edge cases, quotes, and newlines automatically.

## 1. Writing CSV Files

The `csv.Writer` wraps an `io.Writer` (like a file or HTTP response). Because it buffers data for performance, you **must** call `Flush()` when done.

```go
import (
    "encoding/csv"
    "os"
)

func main() {
    file, _ := os.Create("users.csv")
    defer file.Close()

    writer := csv.NewWriter(file)
    defer writer.Flush() // Crucial!

    // Write a single row
    header := []string{"Name", "Age", "City"}
    writer.Write(header)

    // Write multiple rows at once
    data := [][]string{
        {"Alice", "25", "Seattle"},
        {"Bob", "30", "New York"},
        // Notice the comma inside the string! The package will wrap it in quotes automatically.
        {"Charlie", "35", "Washington, D.C."}, 
    }
    writer.WriteAll(data)
}
```

## 2. Reading CSV Files

If the CSV is small, you can read the entire thing into a `[][]string` matrix using `ReadAll()`.

```go
func main() {
    file, _ := os.Open("users.csv")
    defer file.Close()

    reader := csv.NewReader(file)
    
    // Read the whole file into memory
    records, err := reader.ReadAll()
    if err != nil {
        panic(err)
    }

    // records[0] is the header row
    for _, row := range records[1:] {
        fmt.Printf("Name: %s, City: %s\n", row[0], row[2])
    }
}
```

## 3. Streaming Huge CSV Files

If you need to process a 5GB database dump exported as a CSV, `ReadAll()` will crash your server with an Out-Of-Memory error.

Instead, use a loop to stream the file one record at a time. This keeps memory usage flat at a few kilobytes!

```go
import (
    "encoding/csv"
    "fmt"
    "io"
    "os"
)

func main() {
    file, _ := os.Open("massive_dump.csv")
    defer file.Close()

    reader := csv.NewReader(file)
    
    // Optional configuration:
    // reader.Comma = ';' // Change delimiter
    // reader.TrimLeadingSpace = true

    for {
        // Read parses exactly one line
        record, err := reader.Read()
        
        if err == io.EOF {
            break // Reached the end of the file
        }
        if err != nil {
            fmt.Println("Error reading row:", err)
            continue
        }

        // Process the record immediately and let the GC clean it up
        processRow(record)
    }
}
```
