# The `fmt` Package

We have extensively used `fmt.Println` and `fmt.Printf`, but the `fmt` package is capable of much more than just printing to the terminal. 

The true power of `fmt` lies in its integration with the `io.Writer` and `io.Reader` interfaces.

## 1. The Print Family

The `fmt` package has three primary families of print functions:

* **`Print/Println/Printf`**: Writes directly to `os.Stdout` (the standard console output).
* **`Sprint/Sprintln/Sprintf`**: Writes the formatted string to a new `string` variable in memory, returning it instead of printing it.
* **`Fprint/Fprintln/Fprintf`**: Writes the formatted string to **any object that satisfies the `io.Writer` interface**.

## 2. The Power of `Fprintf`

Because `Fprintf` accepts any `io.Writer`, you can use it to write formatted text directly into network connections, HTTP response bodies, or files!

```go
import (
    "fmt"
    "os"
)

func main() {
    // os.Create returns an *os.File, which implements io.Writer!
    file, _ := os.Create("report.txt")
    defer file.Close()
    
    name := "Alice"
    score := 99.5

    // Write directly into the file!
    fmt.Fprintf(file, "User: %s | Score: %.1f\n", name, score)
}
```

### Web Server Example
In a standard Go web server, the `http.ResponseWriter` implements `io.Writer`. You can dynamically format HTML or text straight into the user's browser:

```go
func handler(w http.ResponseWriter, r *http.Request) {
    // Formats and streams directly over the TCP socket to the browser!
    fmt.Fprintf(w, "<h1>Hello, %s!</h1>", r.RemoteAddr)
}
```

## 3. The Scan Family

Just like the Print family, `fmt` provides tools for reading formatted text.

* **`Scan/Scanf`**: Reads from `os.Stdin` (terminal input).
* **`Sscan/Sscanf`**: Reads from a string variable in memory.
* **`Fscan/Fscanf`**: Reads from any object that satisfies the `io.Reader` interface (like a file or network stream).

```go
func main() {
    data := "Alice 25"
    var name string
    var age int

    // Parses the string and extracts the values into the variables via pointers
    fmt.Sscanf(data, "%s %d", &name, &age)
    
    fmt.Println("Name:", name, "Age:", age)
}
```
*Note: While `fmt.Scan` is useful for simple scripts, production systems heavily prefer the `encoding/json` or `bufio` packages for parsing complex data.*
